# yoloDepth 文档与脚本归档目录

本目录包含项目开发过程中产生的各类文档和脚本，已按功能分类归档。

## 📂 目录结构说明

### docs_archive/ - 文档归档
- **八股知识/**: 项目开发过程中积累的面试八股知识点
- **Phase2.5改进文档/**: Phase2.5阶段的改进文档和Bug修复记录
- **Phase3改进文档/**: Phase3阶段的ChannelC2f架构设计文档
- **数据集分析文档/**: VisDrone、UAVDT、COCO等数据集的分析报告
- **项目规划文档/**: 项目规划、性能分析、改进策略等文档

### scripts_archive/ - 脚本归档
- **数据处理脚本/**: 数据集转换、深度图生成、数据分析等脚本
- **测试脚本/**: 各类测试和验证脚本
- **诊断脚本/**: 性能诊断、验证分析等脚本
- **部署脚本/**: 部署检查清单、配置生成等脚本
- **监控脚本/**: 训练监控工具
- **训练脚本_历史/**: 历史训练Shell脚本（已被train_joint.py等替代）

## 🔍 快速查找

### 我想了解...
- **八股知识**: → `docs_archive/八股知识/八股.md`
- **数据集配置**: → `docs_archive/数据集分析文档/`
- **改进历史**: → `docs_archive/Phase2.5改进文档/` 或 `Phase3改进文档/`
- **项目规划**: → `docs_archive/项目规划文档/`

### 我想运行...
- **数据转换**: → `scripts_archive/数据处理脚本/convert_*.py`
- **深度图生成**: → `scripts_archive/数据处理脚本/generate_depths_*.py`
- **数据集验证**: → `scripts_archive/数据处理脚本/verify_*.py`
- **性能诊断**: → `scripts_archive/诊断脚本/diagnose_*.py`

## 📝 使用建议

1. **日常开发**: 使用项目根目录的主要脚本（train_joint.py, val_depth.py等）
2. **查阅文档**: 需要时再到归档目录查找相关文档
3. **历史追溯**: 查看改进文档了解项目演进历史
4. **学习参考**: 八股知识目录适合面试准备和技术学习

## ⚠️ 注意事项

- 归档文件为历史记录，不影响当前项目运行
- 如需使用归档脚本，注意检查路径和依赖是否正确
- 建议定期清理runs/目录的训练结果，保持存储空间
