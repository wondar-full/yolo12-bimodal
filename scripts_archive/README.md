# yoloDepth 脚本归档目录

本目录包含项目开发过程中使用的各类Python和Shell脚本。

## 🔧 使用方法

大部分脚本可以直接运行：

```bash
# 数据处理示例
python scripts_archive/数据处理脚本/convert_uavdt_to_yolo.py

# 诊断脚本示例
python scripts_archive/诊断脚本/diagnose_validation.py
```

## ⚠️ 注意事项

1. **路径问题**: 部分脚本可能使用相对路径，建议从项目根目录运行
2. **依赖检查**: 确保已安装ultralytics等必要依赖
3. **配置更新**: Shell脚本中的路径和参数可能需要根据当前环境调整
4. **历史脚本**: `训练脚本_历史/`目录的脚本已被新脚本替代，仅供参考
